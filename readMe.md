# Task Manager App

This project is a simple Task Manager application built using HTMl, CSS, and Javascript. The goal of this project was to createa dynamic application that allows users to add, view, and manage tasks.

## Technologies Used
- HTML
- CSS
- Javascript

## Features

- Add new tasks with name, category, and deadline
- Display tasks dynamically on the page
- Update task status (In Progress / Completed)
- Filter tasks by status and category
- Clean and responsive UI using Bootstrap

## How It Works 

Tasks are stored in an array and rendered to the DOM using JavaScript. When a new task is added, the array updates and the UI re-renders to display the latest data.

Each task includes:
- Name
- Category
- Deadline
- Status

# Reflection

One of the main challenges I faced during this project was working with dynamic rendering in JavaScript. I had to understand how to take user input, store it in an array, and then display it correctly on the page. At times , small syntax errors such as missing quotation marks or closing tags caused parts of the UI not to render.
To solve these challenges, I focused on breaking the problem down into smaller steps. I tested each part individually. If I had more time, I would improve the application by adding features like deleting and editing tasks. I would also continue improving the styling.
